Privacy Policy

Last updated: March 1, 2026

Overview

This app is an English vocabulary learning tool for middle school students. We take user privacy very seriously.

Data Collection

This app does NOT collect, store, or transmit any personal information. All learning data (progress, game scores) is stored locally on the device only.

Data Storage

This app uses local storage (UserDefaults) to save learning progress and preferences. Data is stored entirely on your device. You can delete it by uninstalling the app.

Third-Party Services

This app does not use any third-party analytics, advertising, or tracking services.

Network Access

This app works completely offline and does not make any network requests.

Children Privacy

This app is suitable for users of all ages, including children under 13. We do not collect any personal information from any users.

Changes

If we make changes to this privacy policy, we will post the updated policy on this page.
